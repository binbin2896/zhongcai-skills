# 预验尸式风险排查清单

> 方法论来源：决策科学的"Premortem"（预验尸）——假设方案已经失败，往前推原因。

## 使用方式

裁决书草稿完成后，关闭正面思维，强制假设：**这份裁决已被对方申请撤销。** 然后问以下问题：

## 排查清单

### 程序防线
- [ ] 对方能否证明"我从未收到开庭通知"？
- [ ] 公告送达前是否穷尽了其他手段（电话+短信+邮寄）？
- [ ] 鉴定材料是否全部经过双方质证签字？
- [ ] 审限届满前是否办理了延长手续并书面告知？
- [ ] 合议记录是否完整、各方签字？

### 实体防线
- [ ] 对方能否证明"裁决的事项不在仲裁协议范围内"？
- [ ] 对方能否证明"我有证据没来得及提交"？
- [ ] 裁决书是否逐项回应了被申请人的答辩意见？
- [ ] 双务合同解除时，是否处理了双方返还义务？

### 仲裁员防线
- [ ] 是否已签署书面披露声明？
- [ ] 庭审中是否与当事人有过私下接触？
- [ ] 裁决书是否亲自起草（非秘书代笔）？

## 评分规则
每个问题回答"是"= 风险点 +1。≥3个"是"需要重点关注。
