# 5大撤裁事由精解与应对策略

> 来源：陶新琴《国内民商事仲裁司法审查》+ 王雪华《仲裁司法审查案例精析》

## 一、违反法定程序（最高频）

| 维度 | 内容 |
|------|------|
| 认定标准 | 仲裁程序违反仲裁规则或法律强制性规定，且可能影响公正裁决 |
| 法院审查口径 | 程序瑕疵一般不撤销（支持仲裁原则），但根本性瑕疵除外 |
| 高频攻击点 | 送达（地址错误/公告不当）、组庭（回避遗漏）、鉴定（材料未质证）、审限（超期未告知） |
| 最佳防御 | 全程留痕——送达记录、庭审笔录、合议记录、审限延长手续，缺一不可 |

## 二、超裁

| 维度 | 内容 |
|------|------|
| 认定标准 | 裁决事项超出仲裁协议范围或超出当事人请求 |
| 可分性原则 | 超裁部分可与未超裁部分区分的，仅撤销超裁部分 |
| 最佳防御 | 裁决书草拟完成后，逐项对照仲裁请求清单，每项裁决主文标注"对应第X项请求" |

## 三、隐瞒足以影响公正裁决的证据

| 维度 | 内容 |
|------|------|
| 认定标准 | 1.证据在仲裁程序中已存在但一方未提交 2.对裁决结果有实质性影响 3.隐瞒方有故意或重大过失 4.申请方已尽合理注意仍未能发现 |
| 认定难度 | 极高（举证责任在申请方，且证明标准严苛） |
| 最佳防御 | 庭审中主动询问"双方是否还有证据需要提交"，记录在案 |

## 四、伪造证据

| 维度 | 内容 |
|------|------|
| 认定标准 | 须有生效刑事判决或司法鉴定证明证据系伪造 |
| 认定难度 | 极高（仅凭一方主张"证据是假的"不足以认定） |
| 最佳防御 | 关键证据要求核对原件，发现涂改/矛盾主动发问并记录 |

## 五、枉法裁决

| 维度 | 内容 |
|------|------|
| 认定标准 | 须有生效刑事判决认定仲裁员构成枉法裁决罪 |
| 认定难度 | 极高（主观故意+行为违法+后果严重，三个条件缺一不可） |
| 最佳防御 | 不与当事人私下接触，遵守廉洁纪律，裁决说理充分 |
