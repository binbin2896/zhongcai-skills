# 程序合规性自查清单

> 来源：写作模板/05-程序合规性自查清单（Feishu）
> 编制依据：陶新琴《国内民商事仲裁司法审查》（南通中院，2024.5.25）

## 10类高频程序违规

| 风险 | 违规类型 | 典型表现 | 防范措施 |
|------|---------|---------|---------|
| 🔴 | 送达程序瑕疵 | 公告送达条件不满足、邮寄地址错误 | 穷尽送达手段并留痕 |
| 🔴 | 组庭不当 | 回避遗漏、仲裁员披露不充分 | 组庭环节设置双重核查 |
| 🔴 | 鉴定程序违规 | 鉴定材料未经质证、鉴定机构未协商 | 建立鉴定材料质证确认表 |
| 🔴 | 缺席审理程序不当 | 公告未满即开庭 | 严格执行公告期规定 |
| 🔴 | 质证程序不完整 | 遗漏交叉质证 | 设证据质证环节清单逐项打勾 |
| 🟡 | 陈述权保障不足 | 打断当事人发言、辩论时间不均等 | 首席主动控制节奏 |
| 🟡 | 释明不当 | 超出引导范围、暗示裁判结果 | 释明限于法律关系和程序事项 |
| 🟡 | 审限管理 | 超审限未办理延长手续 | 到期前30天预警 |
| 🟡 | 笔录签字 | 当事人未核对签字即闭庭 | 留足阅读笔录时间 |
| 🟡 | 合议不规范 | 合议缺席、裁决由秘书代拟 | 合议时间提前确定，裁决书由仲裁员亲自起草 |

## 各阶段自查要点

收案阶段：仲裁协议效力 / 请求范围 / 主体资格 / 仲裁时效 / 反请求期限
组庭阶段：组庭通知 / 回避权利告知 / 主动披露 / 回避申请处理
审理阶段：开庭通知送达 / 公告送达条件 / 陈述权保障 / 举证期限 / 质证完整性
裁决阶段：审限管理 / 合议程序 / 裁决书核阅 / 超裁审查 / 漏裁审查 / 裁决书送达
