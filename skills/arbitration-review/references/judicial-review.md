# 司法审查应对手册

> 来源：写作模板/06-司法审查应对手册（Feishu）
> 编制依据：陶新琴（南通中院）/ 王雪华《中国仲裁司法审查案例精析》

## 三大审查原则

1. 尊重当事人意思自治 — 协议范围内的争议，法院维护仲裁管辖权
2. 支持与监督仲裁并重 — 程序瑕疵不等于撤销理由
3. 有限审查、适度干预 — 法院不审当事人未主张的事由

## 南通中院数据（2018-2023）

- 申请撤销仲裁裁决84件，撤销仅2件（2.3%）
- 申请不予执行35件，部分不予执行3件（8.5%）
- 申请确认仲裁协议效力34件，确认有效80%
- 最高频事由：违反法定程序

## 5大撤裁事由应对

### 1. 违反法定程序（最高频）
12种常见情形：送达瑕疵、组庭不当、陈述权受限、举证期限不公、鉴定违规、超审限、合议庭变更未告知、缺席审理不当、调解不当、书面审理未征同意、未给最后陈述机会、裁决书未送达

### 2. 超裁
防范：裁决书草拟完成后逐项对照仲裁请求清单

### 3. 隐瞒足以影响公正裁决的证据
防范：庭审中主动询问"是否有其他证据需要提交"

### 4. 伪造证据
防范：关键证据要求核对原件，发现涂改矛盾主动发问

### 5. 枉法裁决
防范：裁决说理充分，自由裁量部分说明理由和边界

## 仲裁员执业自我保护

签署书面披露声明 / 庭审记录留痕 / 不合规指令书面拒绝 / 不与当事人私下接触 / 裁决书亲自起草 / 合议意见记录 / 参加年度培训
