# 司法审查应对与撤裁风险防范手册

**编制依据：**
1. 陶新琴《国内民商事仲裁司法审查》（南通中院，2024.5.25）——审查理念+数据
2. 王雪华《中国仲裁司法审查案例精析》三册——案例精析
3. 《仲裁法》第五十八条——撤销裁决法定事由
4. 《民事诉讼法》第二百四十八条（2023修正）——不予执行法定事由

---

## 一、基本认识：司法审查的原则与边界

### 1.1 三大审查原则

| 原则 | 含义 | 对仲裁员的启示 |
|------|------|---------------|
| **尊重当事人意思自治** | 仲裁是基于协议的合意性程序，意思自治是基石 | 凡是当事人协议范围内的争议，法院倾向于维护仲裁管辖权 |
| **支持与监督仲裁并重** | 协助保全、尽量使条款有效、程序瑕疵一般不撤销 | 程序小瑕疵≠撤销理由，但仍需尽力避免 |
| **有限审查、适度干预** | 法院被动审查，不审当事人未主张的事由 | 实体问题（事实认定错误、法律适用错误）不是撤销理由 |

### 1.2 审查结果全景（南通中院2018-2023）

```
                撤销 2 件（2.3%）
                ┌─────────┐
 申请撤销84件 ──┤  驳回   ├── 70件（83.3%）
                │  撤回   ├── 12件（14.2%）
                └─────────┘

                ┌─────────┐
 申请不予执行   │  驳回   ├── 24件（68.5%）
      35件  ────┤  撤回   ├── 8件（23%）
                │不予执行 ├── 3件（8.5%）
                └─────────┘

 确认仲裁协议   │  有效   ├── 80%
      34件  ────┤  无效   ├── 0
                │  撤回   ├── 20%
                └─────────┘
```

**🔍 关键结论：** 仲裁裁决被撤销/不予执行的概率极低（2.3%/8.5%），但程序合规仍是法院审查的核心关注点。

## 二、高频撤裁事由应对

### 2.1 违反法定程序（最高频事由）

**法律依据：** 仲裁法第五十八条第一款第（三）项

**12种常见情形及应对：**

| # | 情形 | 典型表现 | 防范要点 |
|---|------|---------|---------|
| 1 | 未有效送达 | 邮寄地址错误、公告送达条件不满足 | 穷尽送达手段并留痕；电话+短信+邮寄+公告递进使用 |
| 2 | 仲裁庭组成不当 | 仲裁员应回避未回避 | 组庭前完成利益冲突检索（含当事人+代理人的全部关联方） |
| 3 | 当事人陈述权被限制 | 打断辩论、拒绝延期申请、未给予质证机会 | 庭审中主动询问双方是否还有补充，记录在案 |
| 4 | 举证期限设置不公 | 一方获得更长的举证时间 | 双方举证期限一致；延期申请同样标准 |
| 5 | 鉴定程序违规 | 鉴定材料未经质证、鉴定机构未经协商 | 鉴定前召开质证确认会，双方确认送鉴材料 |
| 6 | 超审限未告知 | 审限已过但未办理延长且未通知当事人 | 到期前30天预警，提前办理延长并书面告知 |
| 7 | 合议庭成员变更未告知 | 仲裁员更换后未通知当事人 | 任何变更，立即书面通知并重新征询回避意见 |
| 8 | 缺席审理不当 | 公告未满即开庭、未查明确已送达 | 严格遵守公告期限，确认送达回证后再定开庭时间 |
| 9 | 调解程序不当 | 强制调解、调解中透露裁决倾向 | 调解前明确"自愿"原则，不施压、不预判 |
| 10 | 书面审理未征得同意 | 未经双方同意即决定书面审理 | 书面审理须双方书面同意或有明确约定 |
| 11 | 未给予最后陈述机会 | 辩论结束后直接闭庭，未询问最后意见 | 每次开庭结束前必须询问双方最后意见 |
| 12 | 裁决书未送达或送达不当 | 未按当事人确认地址送达 | 裁决前确认送达地址，优先EMS邮寄+电子送达双保险 |

### 2.2 超裁（第二高频事由）

**法律依据：** 仲裁法第五十八条第一款第（二）项

**认定标准：**
- 裁决的事项超出仲裁协议约定的范围
- 裁决了当事人未提出的请求
- 部分超裁但可与未超裁部分区分的：仅撤销超裁部分

**防范要点：**
1. 裁决书草拟完成后，逐项对照仲裁请求清单
2. 每个裁决主文标注"对应第X项仲裁请求"
3. 部分支持/部分驳回的，明确区分
4. 涉及未签订仲裁协议的第三方时，特别审查管辖权

### 2.3 隐瞒足以影响公正裁决的证据

**法律依据：** 仲裁法第五十八条第一款第（五）项

**认定边界（法院审查标准）：**
- 证据在仲裁程序中已存在，但一方未提交
- 该证据对裁决结果有实质性影响
- 隐瞒方主观上有"隐瞒"的故意或重大过失
- 申请方在仲裁程序中已尽到合理注意义务仍未能发现

**防范要点：**
1. 庭审中主动询问"是否有其他证据需要提交？"
2. 当事人口头提及但未提交关键书证的，令其在规定期限内补充
3. 涉及书面合同/协议缺失的，询问双方是否另有附件或补充协议
4. 对"证据突袭"的应对：给予对方质证机会，不因此剥夺陈述权

### 2.4 伪造证据

**法律依据：** 仲裁法第五十八条第一款第（四）项

**认定标准（严苛）：**
- 必须有生效刑事判决或司法鉴定意见证明证据系伪造
- 仅凭一方主张"证据是假的"不足以认定
- 伪造的证据对裁决结果有实质性影响

**防范要点：**
1. 对关键证据（合同、结算单、函件）要求核对原件
2. 对复印件、传真件、打印件的真实性审慎认定
3. 发现证据存在明显涂改、矛盾之处，主动发问质疑

### 2.5 枉法裁决

**法律依据：** 仲裁法第五十八条第一款第（六）项

**认定标准（极其严苛）：**
- 必须有生效刑事判决认定仲裁员构成枉法裁决罪
- "枉法"要求主观故意+行为违法+后果严重
- 仅凭裁决结果不公不足以认定

**职业保护要点：**
1. 裁决说理充分、论证完整
2. 自由裁量权部分说明理由和边界
3. 不与当事人私下接触，不透露合议意见
4. 遵守仲裁员职业规范和廉洁纪律

## 三、重新仲裁：程序缺陷的补救机制

**适用条件：**
- 法院认为可以通过重新仲裁弥补程序缺陷
- 主要适用于送达瑕疵、陈述权保障不充分等可弥补的程序问题
- 不适用于超裁、无仲裁协议等根本性问题

**操作流程：**
1. 法院通知仲裁庭在一定期限内重新仲裁
2. 原仲裁庭继续审理（不重新组庭）
3. 针对程序缺陷环节补正（重新送达/补充质证/补充告知等）
4. 作出新裁决，原裁决自动失效

## 五、仲裁员执业自我保护清单

| # | 保护措施 | 关键动作 |
|---|---------|---------|
| 1 | 签署书面披露声明 | 收案后3日内完成利益冲突检索并签署 |
| 2 | 庭审记录留痕 | 关键程序节点（释明、询问、当事人承诺）均在笔录中载明 |
| 3 | 不合规指令书面拒绝 | 如被要求违反程序办案，书面说明理由并留存证据 |
| 4 | 不与当事人私下接触 | 所有当事人联系通过秘书或仲裁机构，不私下见面/通话 |
| 5 | 裁决书亲自起草 | 不委托秘书、助手代笔（违反仲裁员管理办法） |
| 6 | 合议意见记录 | 合议内容书面记录，不透露给当事人 |
| 7 | 参加年度培训 | 履行仲裁员守则规定的培训义务 |
| 8 | 及时更新联系方式 | 向仲裁机构报告最新的联系电话和通讯地址 |
