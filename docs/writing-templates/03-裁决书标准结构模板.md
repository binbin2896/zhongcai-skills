# 裁决书标准结构模板

**说明：** 本模板提供简式裁决书、普通程序裁决书、复杂案件裁决书三种范式。根据案件类型选择使用。

---

## 一、裁决书通用结构（"九段式"）

```
┌─────────────────────────────────────────────┐
│ 1. 当事人与代理人基本信息                    │
│ 2. 案件受理与仲裁庭组成情况                   │
│ 3. 仲裁请求、反请求与答辩意见                 │
│ 4. 证据采信与事实认定                        │
│ 5. 仲裁庭意见（争议焦点逐项分析）              │
│ 6. 裁决主文                                │
│ 7. 仲裁费用承担                             │
│ 8. 裁决履行告知                             │
│ 9. 仲裁员签字与日期                          │
└─────────────────────────────────────────────┘
```

---

## 二、简式裁决书模板

> **适用：** 事实清楚、争议不大、适用简易程序的案件

**【首部】**

XXXXXX仲裁委员会
裁决书

（XXXX）X仲裁字第XX号

**申请人：** [姓名/名称]，[性别，民族，出生年月，住址]/[住所地，统一社会信用代码]
法定代表人/负责人：[姓名]，[职务]
委托代理人：[姓名]，[律师事务所/单位/职务]

**被申请人：** [同上格式]

**【引言】**

本会于[受理日期]受理了申请人与被申请人之间的[案由]纠纷仲裁案。经本会主任指定[或：申请人选定、被申请人选定]，由[姓名]担任本案独任仲裁员，[姓名]担任本案办案秘书，适用简易程序，于[开庭日期]在本会开庭审理本案。[申请人/被申请人的委托代理人]到庭参加了仲裁庭审（如有缺席：被申请人经本会书面通知，无正当理由未到庭，本案依法缺席审理）。本案现已审理终结。

**【仲裁请求与答辩】**

申请人称：[仲裁请求和事实理由]

被申请人答辩称：[答辩意见]

（如有反请求，增加：被申请人同时提出反请求……申请人针对反请求答辩称……）

**【仲裁庭认定的事实】**

仲裁庭经审理查明：[清晰的事实叙述，按时间顺序]

**【仲裁庭意见】**

仲裁庭认为：[简要的争议焦点分析 + 法律适用]

**【裁决】**

依据《中华人民共和国[法律名称]》第X条之规定，裁决如下：

一、[具体裁决内容]
二、本案仲裁费人民币[金额]元，由[承担方]承担（已由[预缴方]预缴，由[对方]直接给付[预缴方]）。
三、[履行告知]

本裁决为终局裁决，自作出之日起发生法律效力。

**仲裁员：** [签名]
**日期：** [年]年[月]月[日]日

## 三、普通程序裁决书模板

> **适用：** 标准案件，适用普通程序

**【首部】** （同简式，注明"普通程序"）

**【引言】**

本会于[受理日期]受理了申请人与被申请人之间的[案由]纠纷仲裁案。根据本会仲裁规则的规定，本案适用普通程序。

申请人[方式（选定/指定）]仲裁员[姓名]，被申请人[方式]仲裁员[姓名]，[方式]首席仲裁员[姓名]，于[组庭日期]组成仲裁庭，[姓名]担任办案秘书。

仲裁庭于[开庭日期]开庭审理了本案。[出庭情况]。本案现已审理终结。

**【仲裁请求与答辩】** （同简式，详细列明证据清单）

**【证据采信与事实认定】**

庭审中，双方当事人提交了以下证据：

申请人提交证据：
1. 证据一：[证据名称]，证明：[证明内容]
2. 证据二：[证据名称]，证明：[证明内容]

被申请人对上述证据的质证意见：
1. 对证据一的意见：
2. 对证据二的意见：

被申请人提交证据：
1. 证据一：[证据名称]，证明：[证明内容]

申请人对上述证据的质证意见：
1. 对证据一的意见：

仲裁庭对上述证据的认证意见：
[分别说明采信/不采信的证据、理由]

基于上述采信的证据，仲裁庭认定以下事实：
[事实认定]

**【仲裁庭意见】** （争议焦点逐项分析）

根据双方当事人的仲裁请求、答辩意见及查明的事实，本案的争议焦点归纳如下：

**争议焦点一：[争议焦点名称]**

[详细论证]

**争议焦点二：[争议焦点名称]**

[详细论证]

**【裁决】** （同简式）

## 四、复杂案件裁决书模板

> **适用：** 建设工程案件、群体性案件、法律关系复杂的案件

**结构与普通程序基本一致，增补以下内容：**

### 增补一：案件事实时间轴（在引言后、仲裁请求前）

在引言后增加"案件基本情况"一节，按时间顺序整理关键节点：

> **案件基本情况：**
> [年]年[月]月[日]，双方签订XXXX合同，约定……
> [年]年[月]月[日]，工程开工……
> [年]年[月]月[日]，工程竣工验收合格……
> [年]年[月]月[日]，双方进行结算，确认……
> ……

### 增补二：当事人主张归纳表

在"仲裁请求与答辩"部分，对多请求、反请求、复杂答辩意见用表格归纳：

| 序号 | 申请人主张 | 被申请人抗辩 | 仲裁庭认定 |
|------|-----------|-------------|-----------|
| 1 | | | |
| 2 | | | |
| 3 | | | |

### 增补三：法律适用分段论证

在"仲裁庭意见"中，按"四阶层"结构分层论证：

**第一阶层：法律关系定性**
[明确本案的法律关系性质]

**第二阶层：请求权基础分析**
[分析各项请求权的构成要件 + 本案事实符合性]

**第三阶层：责任划分**
[过失相抵/责任分担的论证]

**第四阶层：损失计算**
[详细展示计算过程]

### 增补四：专业术语解释

在裁决书尾部增加对建工/房地产专业术语的解释说明，确保案外读者能看懂。

## 五、调解/和解撤案裁决书模板

**【首部】** （同简式）

**【引言】**

[基本案情简介]

本案在审理过程中，经仲裁庭主持调解（或：经当事人自行和解），双方当事人自愿达成如下协议：

**【调解协议内容】**

一、[协议内容]
二、[协议内容]

**【裁决】**

根据上述协议，依照《中华人民共和国仲裁法》第四十九条之规定，裁决如下：

一、确认双方当事人达成的调解协议（和解协议）有效。
二、本案仲裁费人民币[金额]元，由[承担方]承担。

本裁决为终局裁决，自作出之日起发生法律效力。

**仲裁员：** [签名]
**日期：** [年]年[月]月[日]日

## 六、写作规范注意事项

| 项目 | 要求 |
|------|------|
| 当事人名称 | 以裁决时营业执照登记名称为准，名称变更在正文载明 |
| 法人/非法人组织 | 法定代表人 vs 负责人——称谓必须与注册登记一致 |
| 反请求标识 | 在称谓后括注"反请求被申请人"/"反请求申请人" |
| 证据引用 | 逐一编号，载明证据名称+内容+证明目的 |
| 质证意见 | 从真实性、合法性、关联性三性归纳 |
| 法律引用 | 条文与认定事实具有关联性，避免泛泛引用 |
| 裁决主文 | 内容完整、用词准确、表述规范、确保可执行 |
| 逻辑层次 | 不出现"合同无效但违约金继续支持"类逻辑矛盾 |
