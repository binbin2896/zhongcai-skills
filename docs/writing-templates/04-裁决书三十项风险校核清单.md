# 裁决书三十项风险校核清单

**说明：** 裁决书草拟完成后，逐项校核。全部🟢方可提交仲裁委员会核阅；存在🔴项应立即修正；存在🟡项应评估后决定。

---

## 一、首部与格式类（5项）

| # | 校核项 | 结果 | 备注 |
|---|--------|------|------|
| 1 | 裁决书文号与案号是否一致？ | □ 🟢 正确 □ 🔴 需更正 | 按文书出具时间生成，非立案号 |
| 2 | 当事人姓名/名称是否与营业执照或身份证一致？ | □ 🟢 一致 □ 🔴 不一致 | 以裁决时登记名称为准 |
| 3 | 法定代表人/负责人的称谓是否准确？ | □ 🟢 准确 □ 🔴 不准确 | 法人用"法定代表人"，非法人用"负责人" |
| 4 | 委托代理人的代理权限是否已核实？ | □ 🟢 已核实 □ 🔴 未核实 | 特别关注一般代理/特别代理 |
| 5 | 裁决书各段格式（字体、行距、编号）是否统一？ | □ 🟢 统一 □ 🟡 需调整 | |

## 二、程序事项类（8项）

| # | 校核项 | 结果 | 备注 |
|---|--------|------|------|
| 6 | 案件受理时间、依据是否写明？ | □ 🟢 已写 □ 🔴 缺失 | |
| 7 | 仲裁庭组成情况是否写清？ | □ 🟢 已写 □ 🔴 缺失 | 指定/选定的方式和日期 |
| 8 | 是否写明仲裁员回避及重新组庭情况（如有）？ | □ 🟢 已写 □ 🟡 不适用 | |
| 9 | 开庭时间、出庭人员是否核对无误？ | □ 🟢 正确 □ 🔴 有误 | 多次开庭的需写明每次情况 |
| 10 | 缺席审理的，是否写明公告送达/通知情况？ | □ 🟢 已写 □ 🟡 不适用 | |
| 11 | 管辖权异议是否处理并写明决定？ | □ 🟢 已处理 □ 🔴 未处理 | |
| 12 | 是否存在未被回应的当事人程序性异议？ | □ 🟢 已全部回应 □ 🔴 遗漏 | |
| 13 | 是否存在超审限情形？是否已办理延长手续？ | □ 🟢 正常 □ 🟡 超审限已延长 □ 🔴 超审限未办 | |

## 三、仲裁请求与答辩类（4项）

| # | 校核项 | 结果 | 备注 |
|---|--------|------|------|
| 14 | 是否完整列明了申请人的仲裁请求？ | □ 🟢 完整 □ 🔴 遗漏 | 逐项核对请求书 |
| 15 | 是否有反请求？反请求是否已完整列明并处理？ | □ 🟢 已列 □ 🔴 遗漏 □ 🟡 不适用 | |
| 16 | 被申请人的答辩意见是否被摘要归纳？ | □ 🟢 已归纳 □ 🔴 未归纳 | 归纳不可扭曲原意 |
| 17 | 是否存在超出仲裁协议范围的请求/反请求？ | □ 🟢 范围内 □ 🔴 超裁 | 超裁部分需说明理由且可区分 |

## 四、事实认定与证据类（6项）

| # | 校核项 | 结果 | 备注 |
|---|--------|------|------|
| 18 | 证据是否逐一举证质证并在裁决书中体现？ | □ 🟢 已体现 □ 🟡 可简化 | 简易程序可归纳，不可省略 |
| 19 | 质证意见是否从真实性、合法性、关联性三性归纳？ | □ 🟢 已归纳 □ 🟡 不够清晰 | |
| 20 | 仲裁庭对证据的认证是否说明了采信/不采信的理由？ | □ 🟢 已说明 □ 🔴 未说明 | 简单罗列"与本案有关，予以采信"不合格 |
| 21 | 事实认定是否基于采信的证据？有无推定或推断？ | □ 🟢 有据 □ 🟡 有推断（已说明理由） | |
| 22 | 合同效力的认定是否分析了双方过错？ | □ 🟢 已分析 □ 🔴 未分析 | 合同无效时必须分析 |
| 23 | 司法鉴定意见是否经质证？ | □ 🟢 已质证 □ 🔴 未质证 | 未经质证的鉴定报告不得作为认定依据 |

## 五、仲裁庭意见与说理类（5项）

| # | 校核项 | 结果 | 备注 |

| 26 | 法律适用是否与认定的事实具有关联性？ | □ 🟢 关联 □ 🔴 不关联 | |
| 27 | 自由裁量权的运用是否说理充分？ | □ 🟢 已说理 □ 🟡 需加强 | |
| 28 | 双务合同解除时，是否处理了双方返还义务？ | □ 🟢 已处理 □ 🔴 未处理 | 避免只判退款不问返还 |
