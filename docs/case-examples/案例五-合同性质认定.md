# 案例五：合同性质认定——ALC预制墙板施工合同

**问题类型：** 🟡 法律关系定性

---

## 一、基本案情

申请人与被申请人于2020年11月11日签订了《项目施工合同（ALC预制墙板）》，工程内容为**ALC预制墙板的供货和安装**，计价方式为固定单价。

申请人是**劳务公司**，没有相应的建筑施工企业资质。

## 二、裁决书原稿

仲裁庭认为，买卖合同是出卖人转移标的物所有权于买受人，买受人支付价款的合同。本案中，从案涉合同约定的承包范围来看，**合同的主要目的是转移标的物的所有权**，而安装义务只是附随义务，故案涉《项目施工合同（ALC预制墙板）》虽然名为项目施工合同，**其性质实质应认定为买卖合同**。该合同系双方真实意思表示，合法有效。

## 三、原稿存在的问题

将ALC预制墙板供货+安装定性为买卖合同，**忽视了ALC墙板在建筑工程中的技术属性和功能定位**。

## 四、相关法律依据

**《建设工程质量管理条例》第二条**：本条例所称建设工程，是指土木工程、建筑工程、线路管道和设备安装工程及装修工程。

## 五、修改后的仲裁庭意见

本案中，案涉《项目施工合同（ALC预制墙板）》名为"项目施工合同"，从合同约定的内容来看，计价方式为固定单价，工程内容为ALC预制墙板的供货和安装。

**ALC墙板（蒸压轻质混凝土板）属于装配式结构工程的一种施工材料和工艺**，在本案申请人的施工中，将该ALC预制墙板作为内外隔墙使用，**内外隔墙等二次结构属于主体结构工程施工范围**，故案涉合同的性质应认定为**建设工程施工合同**。

申请人系劳务公司，没有相应的建筑施工企业资质，故其作为承包人签订的案涉《项目施工合同（ALC预制墙板）》违反了法律的强制性规定，**应认定为无效**。由于本案的双方当事人对案涉合同的无效均有过错，故双方应对案涉合同的无效承担同等责任。

## 六、案例价值

| 维度 | 要点 |
|------|------|
| 裁判思路 | 合同定性不能只看名称和表面约定，要看**技术属性和工程实践** |
| 区分要点 | 买卖合同（仅转移所有权）vs 建设工程施工合同（含施工+安装） |
| 实务启发 | ALC墙板=装配式结构，属主体结构工程范围——这是行业共识，裁判者应知悉 |
