# 案例六：双务合同解除时的释明义务

**问题类型：** 🟡 程序义务 / 释明权行使

---

## 一、基本案情

2019年6月14日，申请人A与被申请人B公司签订《车辆买卖合同书》一份，约定A向B公司购买某型号手动中配汽车一辆，购车金额65,800元。

车辆交付后，申请人A因倒车影像等问题多次至4S店维修，但未得到解决。申请人A遂提起仲裁，请求**解除**与被申请人B公司签订的《车辆买卖合同书》，并**全额退还购车款**。

对此，被申请人B公司提出车辆不符合退车条件，其**不同意解除合同**。

## 二、裁决书初稿存在的问题

**裁决书初稿：** 案涉合同虽不具备解除的条件，但双方当事人均已无继续履行案涉合同意愿，故双方在合同项下的权利义务应予终止，并裁决在扣除合理使用补偿费后，由被申请人退还申请人购车款58,824.87元。

**问题：**
1. 本案系**双务有偿合同**，仲裁庭仅处理退款问题，而未对车辆予以处置，是否导致当事人之间的利益失衡？
2. 仲裁庭需行使**释明权**，告知被申请人B公司提出同时履行抗辩。

## 三、相关法律依据

**《民法典》第五百六十六条第一款**：合同解除后，尚未履行的，终止履行；已经履行的，根据履行情况和合同性质，当事人可以请求恢复原状或者采取其他补救措施，并有权请求赔偿损失。

**《全国法院民商事审判工作会议纪要》第36条**（释明义务核心规定）：
- 在双务合同中，要防止机械适用"不告不理"原则
- 应向原告释明变更或者增加诉讼请求，或者向被告释明提出**同时履行抗辩**
- 尽可能**一次性解决纠纷**
- 第一审人民法院未予释明的，第二审人民法院可以直接释明并改判

**《九民纪要》第49条**：合同解除时，一方依据违约责任条款请求另一方承担违约责任的，人民法院依法予以支持。双务合同解除时人民法院的释明问题，参照本纪要第36条的相关规定处理。

## 四、修改后的仲裁庭意见

参照《民法典》第五百六十六条第一款规定：合同解除后，已经履行的，根据履行情况和合同性质，当事人可以请求恢复原状。

根据前述仲裁庭意见，仲裁庭已认定双方当事人在案涉合同项下的权利义务终止，**双方互负返还义务**。因此，仲裁庭认定在扣除合理使用补偿费后，被申请人还应退还申请人购车款58,824.87元，**申请人在收取退款的同时，也应将案涉某型号汽车返还给被申请人**。同时，仲裁庭在裁决主文中明确**退款与返还车辆双方应同时履行**。

## 五、案例价值

| 维度 | 要点 |
|------|------|
| 裁判思路 | 双务合同解除不能只判退款不问返还——否则导致利益失衡 |
| 程序义务 | 仲裁庭应主动释明，引导当事人提出同时履行抗辩 |
| 写作规范 | 裁决主文应明确"同时履行"关系，确保裁决的可执行性 |
