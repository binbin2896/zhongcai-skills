# 案例一：仲裁条款瑕疵

**问题类型：** 🔴 管辖权认定

---

## 一、基本案情

申请人A与被申请人B签订了《租赁合同》一份，其中关于争议条款的约定如下：**"因执行本合同发生争议，由争议双方协商解决，协商不成，双方同意向南通经济合同仲裁委员会申请仲裁。"**

## 二、裁决书原稿存在的问题

仲裁庭意见中**未对仲裁条款的效力进行认定**。

## 三、相关法律依据

1. **《仲裁法》第十六条**：仲裁协议包括合同中订立的仲裁条款和以其他书面方式在纠纷发生前或者纠纷发生后达成的请求仲裁的协议。仲裁协议应当具有下列内容：（一）请求仲裁的意思表示；（二）仲裁事项；（三）选定的仲裁委员会。
2. **《最高人民法院关于适用〈中华人民共和国仲裁法〉若干问题的解释》第三条**：仲裁协议约定的仲裁机构名称不准确，但能够确定具体的仲裁机构的，应当认定选定了仲裁机构。
3. **同上解释第六条**：仲裁协议约定由某地的仲裁机构仲裁且该地仅有一个仲裁机构的，该仲裁机构视为约定的仲裁机构。该地有两个以上仲裁机构的，当事人可以协议选择其中的一个仲裁机构申请仲裁；当事人不能就仲裁机构选择达成一致的，仲裁协议无效。

## 四、仲裁庭意见（修改稿）

仲裁庭认为：关于《租赁合同》中约定的仲裁条款的效力问题。根据《最高人民法院关于适用〈中华人民共和国仲裁法〉若干问题的解释》第三条规定："仲裁协议约定的仲裁机构名称不准确，但能够确定具体的仲裁机构的，应当认定选定了仲裁机构。"第六条规定："仲裁协议约定由某地的仲裁机构仲裁且该地仅有一个仲裁机构的，该仲裁机构视为约定的仲裁机构……"

本案中，双方当事人在案涉《租赁合同》约定"因执行本合同发生争议，由争议双方协商解决，协商不成，双方同意向南通经济合同仲裁委员会申请仲裁"，双方约定的仲裁机构名称不准确，但因**南通市仅有一家仲裁机构，即本会**，故本会应视为双方当事人选定的仲裁机构。

因此，根据《中华人民共和国仲裁法》第十六条的规定，案涉《租赁合同》中约定的仲裁条款有效，本会对双方因案涉合同所产生的纠纷具有仲裁管辖权。

## 五、案例价值

| 维度 | 要点 |
|------|------|
| 裁判思路 | 仲裁条款效力必查，不能遗漏——无论当事人是否提出异议 |
| 法律适用 | 名称不准确但可确定的，依司法解释第三、六条认定有效 |
| 写作规范 | 裁决书必须有"仲裁协议效力认定"段落，这是管辖权的基石 |
