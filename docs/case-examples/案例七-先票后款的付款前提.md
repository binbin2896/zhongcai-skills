# 案例七："先票后款"付款前提的认定

**问题类型：** 🟡 合同义务履行顺序

---

## 一、基本案情

2021年3月，申请人A公司与被申请人B公司签订《设计合同》一份，约定由申请人A公司负责某项目地库品质深化设计工作，合同约定设计费116,000元。

在约定了分四期支付设计费的同时，双方在合同中还约定：**被申请人B公司每次付款前，申请人A公司均应先开具相应金额的增值税专用发票，否则被申请人B公司有权不支付，且不承担任何责任**，但申请人A公司仍需按照合同约定履行相关义务。

庭审中，双方共同确认申请人A公司向被申请人B公司**交付了设计成果**，而**未开具任何发票**。

## 二、裁决书初稿存在的问题

**裁决书初稿：** 对于被申请人提出的因申请人未开具相应金额的增值税专用发票，故付款条件未成就的抗辩，仲裁庭不予采信。申请人虽尚未开具增值税专用发票，但这是被申请人未及时履行前期付款义务导致，被申请人不得以"先票后款"约定为由拒绝付款。

**问题：**
1. 开具发票系申请人的**从合同义务**，而付款系被申请人的**主合同义务**——在双方存在特别约定的情况下，被申请人能否行使**先履行抗辩权**？
2. 如被申请人的先履行抗辩权成立，是径行裁决还是**变通处理**为妥？

## 三、相关法律依据

**《最高人民法院关于适用〈中华人民共和国民法典〉合同编通则若干问题的解释》第三十一条**：
"当事人互负债务，一方以对方没有履行**非主要债务**为由拒绝履行自己的主要债务的，人民法院不予支持。但是，对方不履行非主要债务致使不能实现合同目的或者**当事人另有约定的除外**。"

## 四、庭后处理与最终结果

庭后，申请人向仲裁庭补充提交**《电子发票（增值税专用发票）》及微信聊天记录截图**，证明：申请人已开具案涉合同项下11,600元的全额增值税专用发票，并于2023年12月5日交付被申请人。

仲裁庭在收到上述补充证据后随即发送给被申请人。被申请人对上述证据发表书面质证意见为：对该证据的真实性、合法性以及关联性予以认可。

**最终仲裁庭意见：** 本案申请人已向被申请人交付了设计成果，其也已向被申请人开具了11,600元的增值税专用发票，被申请人的付款条件已经成就，付款期限也已届满。因此，仲裁庭对申请人要求被申请人支付设计费的仲裁请求予以支持。

## 五、案例价值

| 维度 | 要点 |
|------|------|
| 裁判思路 | "先票后款"条款原则上有效（当事人另有约定除外），不宜直接否定 |
| 操作技巧 | 发现程序漏洞后，先给当事人机会补充证据（庭后补开+交付），比直接裁判更稳妥 |
| 区分要点 | 主合同义务（付款）vs 从合同义务（开票）——但当事人明确约定后，抗辩权规则优先适用 |
