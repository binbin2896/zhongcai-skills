# 案例三：审理范围

**问题类型：** 🔴 管辖权认定

---

## 一、基本案情

2013年9月15日，申请人A与被申请人B签订了《施工合同》，约定案涉工程发包给申请人施工；约定工程承包范围：研发楼和三栋厂房桩基、土方、砌筑、钢筋混凝土、屋面及防水、楼地面、抹灰、油漆涂料、门窗、幕墙、钢雨棚、室内水电、消防工程。

在案涉《施工合同》的履行过程中，申请人**又承包施工了室外配套工程**，但双方**未就室外配套工程另行签订书面合同**。

## 二、裁决书原稿存在的问题

申请人主张的工程款包含室外配套工程的工程款，**在被申请人缺席的情况下，对室外配套工程的管辖权未评价**。

## 三、相关法律依据

**《仲裁法》第四条**：当事人采用仲裁方式解决纠纷，应当双方自愿，达成仲裁协议。没有仲裁协议，一方申请仲裁的，仲裁委员会不予受理。

## 四、补救措施与仲裁庭意见

**庭后**：联系双方就配套工程**补充达成仲裁协议**。

仲裁庭意见：
仲裁庭注意到，案涉《施工合同》所涉工程为研发楼、厂房土建、水电安装工程。在案涉《施工合同》的履行过程中，申请人又承包施工了室外配套工程，但双方未就室外配套工程另行签订书面合同。

仲裁庭认为：申请人庭后补充提交了**加盖双方公司公章的《情况说明》**，其中明确载明"申请人承建被申请人开发的……研发楼、厂房及室外配套工程项目……双方愿意将上述所有工程项目及工程款等全部争议提交给贵委处理"。被申请人对该《情况说明》的真实性予以认可。

因此，本会对双方当事人在案涉《施工合同》和室外配套工程项下的纠纷均具有管辖权。

## 五、案例价值

| 维度 | 要点 |
|------|------|
| 裁判思路 | 未签合同的新增工程不当然落入原仲裁协议范围，需当事人补充确认 |
| 操作要点 | 发现管辖漏洞后应立即庭后补充——让当事人补签情况说明或补充仲裁协议 |
| 写作规范 | 管辖权问题应在仲裁庭意见开头即处理，不能等实体认定后再回头补 |
