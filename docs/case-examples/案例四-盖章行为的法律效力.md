# 案例四：盖章行为的法律效力

**问题类型：** 🔴 合同效力认定 / 代理权判断

---

## 一、基本案情

2013年7月，发包人与被申请人（承包人，落款处加盖被申请人印章、委托代理人处沈某签名）就某工程签订《施工总承包合同》。

2014年1月21日，申请人（乙方）与被申请人（甲方，落款处加盖"**工程项目部**"印章，王某华签名）就11号、18号楼整体工程签订《分包协议》。

2015年12月，申请人施工的11号、18号楼工程竣工验收合格。

2023年3月，申请人与沈某共同出具《结算说明》。

**申请人仲裁请求：**
①被申请人支付工程款人民币12,057,415.81元及利息
②案件受理费、保全费等由被申请人负担

**被申请人答辩意见：** 案涉《分包协议》不是其与申请人签订，王某是沈某聘请的现场管理人员，**不是被申请人的工作人员**，不构成职务行为及表见代理。

## 二、争议焦点

王某的签名行为是否对被申请人构成**职务行为或表见代理**，以及该《分包协议》是否对被申请人发生法律效力。

## 三、裁判思路——"看人不看章"

根据**《九民纪要》第41条**关于盖章行为法律效力的规定：**"看人不看章"**——盖章行为的本质在于表明行为人从事的是职务行为，而从事职务行为的前提是盖章行为人不仅须是单位的工作人员，而且还需要享有代表权或代理权。

有代表权或代理权的人盖章确认的合同，自然对单位具有约束力。而无代表权或代理权的人加盖的公章，**即便是真章，也不能产生合同有效的预期效果**。

因此，项目部印章之于案涉《分包协议》的效力，**关键不在印章的真假，而在盖章之人王某有无代理权**。

## 四、仲裁庭意见分析

**1. 关于是否构成职务代理**

职务代理须满足：①代理人是法人工作人员；②代理人实施的必须是其职权范围内的事项；③必须以法人名义实施法律行为。

本案中，申请人与被申请人均确认**王某是案外人沈某聘请的现场管理人员**，而非被申请人的工作人员——不符合职务代理的构成要件。

**2. 关于是否构成表见代理**

表见代理须满足：①行为人以被代理人名义实施无权代理行为；②客观上存在使相对人相信行为人具有代理权的理由；③**相对人主观上须为善意且无过失**。

法院认为：
- 客观上：王某持有并保管项目部印章，但该印章是沈某刻制的，不能推定其具有代理权外观
- 主观上：案涉款项超过50,000,000元，申请人未审查王某是否有授权委托书——**未尽合理审查义务**
- **结论：不构成表见代理**

**3. 关于协议是否对被申请人发生效力**

虽然王某的签约行为系无权代理，但被申请人通过以下行为**事后追认**了该协议：
- 2015年5月18日出具的《委托收款书》中**书面确认**申请人为11号、18号楼分包人
- 2020年诉讼中依据申请人施工内容向发包人主张工程款
- **九年之久从未提出异议，不合常理**
- 中级人民法院在《民事裁定书》中初步认定协议已成立并实际履行

**结论：协议因被申请人的事后追认而发生效力。**

## 五、案例价值

| 维度 | 要点 |
|------|------|
| 裁判思路 | **"看人不看章"**——印章真假不是合同效力的关键，代理权才是 |
| 区分要点 | 职务代理 vs 表见代理 vs 无权代理的追认——三种不同路径，构成要件各异 |
| 写作规范 | 判断顺序：先问有无代理权→再问能否构成表见代理→最后看是否被追认 |
| 实务启发 | 巨额合同（超5000万）必须审查签约人授权文件，否则可能被认定为未尽合理注意义务 |
