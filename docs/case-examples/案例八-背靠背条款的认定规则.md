# 案例八："背靠背"条款的认定规则

**问题类型：** 🔴 合同条款效力认定

---

## 一、基本案情

2021年7月10日，申请人A与被申请人B签订了《分包合同》一份，其中关于付款条件的约定如下：本工程的劳务报酬的付款条件分为：
（1）本工程无预付款，无中间支付；
（2）施工完成后人员退场付至合同价的40%；
（3）**竣工验收后项目大结算完成付至结算价的95%**。

后双方当事人因合同的履行产生争议，申请人遂向本会申请仲裁，请求被申请人向申请人支付工程款以及利息损失。

**被申请人抗辩：** 所谓"大结算"是指**被申请人与总包单位的结算**，目前双方仍未结算，该项付款条件仍未成就。

## 二、裁决书初稿存在的问题

**没有直接回应被申请人的该项抗辩。**
裁决书仅载明："案涉《分包合同》第二条第二款第3项约定：'竣工验收后项目大结算完成付至结算价的95%'。被申请人提交的补充证据四《竣工验收备案表》证明案涉工程竣工验收备案登记时间为2021年11月30日，申请人请求支付逾期付款利息的起算时间自2022年1月11日起，对此仲裁庭予以认可。"

**问题：完全回避了"大结算未完成"的核心抗辩。**

## 三、相关法律依据

**《民法典》第六条**：民事主体从事民事活动，应当遵循公平原则，合理确定各方的权利和义务。

**《民法典》第一百五十三条**：违反法律、行政法规的强制性规定的民事法律行为无效。

**法释〔2024〕11号《最高人民法院关于大型企业与中小企业约定以第三方支付款项为付款前提条款效力问题的批复》**：
- 大型企业与中小企业约定以收到第三方向其支付的款项为付款前提的，因其内容违反《保障中小企业款项支付条例》第六条、第八条的规定，**人民法院应当认定该约定条款无效**
- 认定条款无效后，应结合行业规范、双方交易习惯等，合理确定大型企业的付款期限及相应的违约责任

**《保障中小企业款项支付条例》第六条、第八条**：机关、事业单位和大型企业不得要求中小企业接受不合理的付款期限、方式和条件。

## 四、仲裁庭意见（修改稿）

仲裁庭认为：该约定系**"背靠背"条款**。被申请人援引该条款对抗申请人要求支付工程款的请求，需对其履行了相应的**合同附随义务**承担举证责任：

1. **订约告知义务**：被申请人应当以明示或其他适当方式向申请人告知总包合同价款支付的约定，但被申请人**未提交证据**证明其履行了订约告知义务；
2. **履约告知义务**：被申请人应当就发包方对工程进度款、结算款的给付情况以及履约过程中其与发包方产生的争议情况及时告知申请人，被申请人**未提交证据**证明其履行了履约告知义务；
3. **履约注意义务**：被申请人应当在履约过程中**积极向发包人主张工程款**，被申请人**未提交证据**证明其已经通过各种方式积极向发包方主张权利。

因此，被申请人应承担对其不利的法律后果，即**不能援引该条款来对抗申请人要求支付工程款的请求**。仲裁庭对于被申请人的该项抗辩不予采纳。

## 五、案例价值

| 维度 | 要点 |
|------|------|
| 裁判思路 | "背靠背"条款不当然无效，但援引方须证明三项附随义务已履行 |
| 三项义务 | 订约告知→履约告知→积极主张——缺一不可，举证责任在被申请人 |
| 新规影响 | 法释〔2024〕11号后，大型企业对中小企业的"背靠背"条款可能直接认定无效 |
| 写作规范 | 当事人抗辩必须逐项回应——回避等于程序瑕疵 |
